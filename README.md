## Installation

The recommended way is to install via Cocoapods:

Add into your Podfile:

```ruby
pod 'iosenv', :git => 'https://github.com/stanislaw/iosenv'
```

And run 

```
pod update
```

or you can just add `iosenv/` folder or `iosenv.h` file to your project.
